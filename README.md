## installatie git repo

- Clone de repo, door in de terminal dit commando uittevoeren.

```shell
git clone https://gitlab.com/ikdoeict/miro.matton/2021wmmad-lets.git
```

# installatie

- instaleren van dependencies

```shell
npm install
```

- app starten

```shell
npm start of expo start
```
